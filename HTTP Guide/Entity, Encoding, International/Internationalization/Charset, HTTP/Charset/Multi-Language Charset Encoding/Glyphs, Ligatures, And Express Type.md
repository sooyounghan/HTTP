-----
### 글리프, 연자 그리고 표현 형태
-----
1. 글리프와 문자를 헷갈려서는 안 됨
   - 문자는 유일하고 추상화된 언어 요소
   - 글리프는 각 글자를 그리는 특정한 방법
   - 각 문자는 미적인 양식과 스크립트에 따라 여러 가지 글리프를 가짐 (즉, 글리프란 글꼴이나 사소한 미적 양식에 의존하지 않는, 글자에 내재된 모양새)

2. 많은 필기체와 활자체가 인접한 글자들이 부드럽게 이어지는 연자(Ligature)를 지원 (문자와 혼동하지 말것)

3. 일반적인 규칙은, 글리프 하나를 다른 것으로 바꾸었을 때 텍스트 의미가 바뀐다면, 두 글리프들은 서로 다른 글자
   - 아니라면, 그것들은 모양만 다를 뿐 같은 글자
