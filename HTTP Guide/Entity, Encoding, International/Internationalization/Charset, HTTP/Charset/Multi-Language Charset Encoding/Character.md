### 문자
-----
1. 쓰기의 기본적인 구성 요소
2. 하나의 문자는 하나의 알파벳 글자, 숫자, 구두점, 표의문자, 수학 기호 혹은 그 외 다른 쓰기 기본 단위 표현
3. 문자는 글꼴이나 스타일에 독립적
