### 표준화된 MIME Charset 값
-----
1. 특정 문자 인코딩과 특정 코딩된 문자집합의 결합을 MIME Charset
2. HTTP는 표준화된 MIME Charset 태그를 Content-Type과 Accept-Charset 헤더에 사용
   - MIME Charset 값은 IANA에 등록되어 있음
