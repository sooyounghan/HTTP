### 일반 헤더
-----
1. 메세지에 대한 아주 기본적인 정보 제공
2. 예를 들어, 요청 메세지나 응답 메세지를 작성할 때, 요청에서나 응답에서나 메세지가 생성된 일시는 같은 의미를 가지므로, 이런 종류의 정보를 제공하는 헤더는 양쪽 메세지 모두에 대해 일반적
